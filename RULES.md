# Правила по инструментам

Полный источник истины проекта: [`PROJECT_MASTER_v59.md`](PROJECT_MASTER_v59.md).

Этот файл оставлен как короткий справочник правил по конкретным инструментам.
Они должны выводиться отдельной сноской при будущей проверке журнала и
статистики. Правила не меняют расчёты или доступность сделки автоматически,
пока не будет отдельного согласования.

## SILVER

### SILVER-001 — Стата + К-3

- Возможные уровни: 38 / 23; К-3 использует один фактический вход.
- Стоп: за К2 с буфером +/- 0.200.
- Цели: К-3, как в Стате.

### SILVER-002 — Стата + И

- Стоп: за 14 с буфером +/- 0.200.

### SILVER-003 — В+И

- Входы И: 50 / 23.
- Стоп: за 14 с буфером +/- 0.200.

### SILVER-004 — Диапазон

- Уровни сетки: 38 / 14 / -15.
- Стоп: за -38.
- Цели: цели диапазона.

### SILVER-005 — ВТ5

Формулировка пользователя:

> Для серебра стратегия ВТ5: вход по СЕРЧам и лимиткам + Крюк + МакД,
> строго по нему + пересечение КЗ и Крюка по направлению движения.

Дополнительное условие: пересечение КЗ и Крюка по направлению движения,
допуск +/- 3 бара.

Статус: зафиксировано для будущей сноски отчёта и напоминания при проверке
журнала и статистики. В приложении пока не применяется как автоматическая
проверка или ограничение.
