# Правила по инструментам

Этот файл хранит согласованные правила пользователя по конкретным инструментам.

Пока правила из этого файла не меняют расчёты, риск, журнал или доступность
сделки. Они должны выводиться одной отдельной сноской при будущей проверке
журнала и статистики по выбранному инструменту.

## SILVER

### SILVER-001 — правило для стратегии «ВТ5»

Формулировка пользователя:

> Для серебра стратегия ВТ5: вход по СЕРЧам и лимиткам + Крюк + МакД,
> строго по нему + пересечение КЗ и Крюка по направлению движения.

Статус: зафиксировано для будущей сноски отчёта и напоминания при проверке
журнала и статистики. В приложении пока не применяется как автоматическая
проверка или ограничение.
